# Introduction

[**hrmp**][hrmp] is a high-resolution music player for Linux.

## Features

* FLAC (44.1kHz/16bits) (Stereo)
* WAV (44.1kHz/16bits) (Stereo)
* By-pass PulseAudio for native bit-stream

## Platforms

The supported platforms are

* [Fedora][fedora] 42+
* [RockyLinux][rocky] 9
